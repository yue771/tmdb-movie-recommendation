# 🎬 Movie Data Analysis & Recommendation System

## 📌 Project Overview
This project analyzes movie data from TMDB and builds a recommendation system based on movie genres.

## 📊 Key Analysis
- Genre distribution (Drama, Comedy, Action dominate)
- Language distribution (English dominates)
- Movie release trends over time
- Rating distribution
- Revenue vs Rating analysis

## 🤖 Recommendation System
- Content-based filtering
- Built using CountVectorizer and cosine similarity
- Recommends movies based on genre similarity

## 🛠️ Tech Stack
- Python (Pandas, NumPy)
- Matplotlib
- Scikit-learn

## 📈 Key Insights
- Most movies fall between ratings 6–8
- Revenue is weakly correlated with rating
- Genre plays a major role in movie similarity

## 😊 DATASET
- Full TMDB Movies Dataset 2024 (1M Movies)
